Server Connection

How to Join

Server Address
recktsdragonhorizon.aternos.me:20126
⚠This server runs on Aternos — it must be manually turned on before players can connect. Ask the server owner (Reckt) to start it up.
◆In Minecraft, go to Multiplayer → Add Server, paste the address above, and save. It will appear in your server list ready to join whenever the server is online.
◆Make sure all mods are installed before connecting. Joining without the correct mods will result in a connection error or crash.
Installation

Step-by-Step Setup

01
Download and install Minecraft Java Edition 1.20.1 from the official Minecraft launcher if you haven't already.
02
Download and run the Forge 1.20.1 installer (build 47.4.x) from the official Forge website. Select "Install client" when prompted.
03
Click the Download from Google Drive button above and download the mods folder ZIP.
04
Extract the ZIP and copy the mods folder into your .minecraft directory. On Windows this is usually at %appdata%\.minecraft.
05
Open your Minecraft launcher, select the Forge 1.20.1 profile. In settings, set JVM arguments to allocate RAM: -Xmx8G -Xms4G
06
Launch the game — first load takes 3–5 minutes. Once at the main menu, go to Multiplayer and connect to the server address above.
