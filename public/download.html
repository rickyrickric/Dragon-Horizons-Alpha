Installation

Step-by-Step Setup

01
Download and install Minecraft Java Edition 1.20.1 from the official Minecraft launcher.
02
Download and run the Forge 1.20.1 installer (build 47.4.x). Select "Install client" when prompted.
03
Verify your access above, then click the Download from Google Drive button.
04
Extract the ZIP and copy the mods folder into your .minecraft directory. On Windows: %appdata%\.minecraft
05
Select the Forge 1.20.1 profile. In JVM arguments, allocate RAM: -Xmx8G -Xms4G
06
Launch the game — first load takes 3–5 minutes. Go to Multiplayer and connect to the server address above.
