# Dragon Horizons

A Soulslike Minecraft Modpack for Forge 1.20.1

## Overview

Dragon Horizons transforms Minecraft survival into a challenging, rewarding experience inspired by Soulslike games. Mobs scale with distance from spawn, gear progression is locked behind skill gates, and legendary weapons only drop from defeated enemies.

## Features

- **Soulslike Combat**: Posture mechanics, bleed buildup, weapon arts, 9 original bosses
- **Living Difficulty**: AutoLeveling scales mob stats by exploration distance
- **Earned Progression**: 8-discipline skill system gates all gear tiers
- **Rich World**: Ice and Fire, Alex's Mobs, Alex's Caves, Mobs of Mythology
- **Multiplayer Ready**: All configs tuned for shared server play

## System Requirements

- **Minecraft**: Java Edition 1.20.1
- **Mod Loader**: Forge 47.4.x
- **Java**: Java 17 or higher
- **RAM**: 6-8 GB allocated
- **Disk Space**: 4 GB free
- **GPU**: GTX 970 / RX 480 (minimum)

## Installation

1. Install Minecraft Java Edition 1.20.1
2. Install Forge 1.20.1 (build 47.4.x)
3. Download the mods folder from our website
4. Extract and copy to `.minecraft/mods`
5. Launch Minecraft with the Forge profile
6. Connect to the server
